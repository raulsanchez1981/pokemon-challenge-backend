POKEMON TOUR

La Fiebre Pokémon

El pequeño Alejandro Infantes Herrera (nombre ficticio) es un viciata de Pokémon. Se ha pasado desde la vetusta y monocroma versión de Game Boy hasta el Pokémon Go. Ya no sabe qué hacer, y ha decidido hacerse su propia pokédex. Como se los sabe todos de memoria, tiene más información que nintendo y es un pocafaena, los datos los picará él a mano. Aunque su proyecto es un tanto ambicioso.

Va a empezar por un MVP con el que podrá:

  • Listar todos los pokémons
  • Crear, editar o borrar pokémons
  • Ir a la ficha de un pokémon
  • Marcar un pokémon como favorito

Para lograr este trabajo, ya se ha realizado el trabajo no visible. Se ha creado una aplicación API REST donde descansan todos los métodos necesarios para que sean consultados. Para poder acceder al código, documentación, etc:

  • Documentación de la aplicación y pruebas con Swagger
  • Código de la aplicación en Git Hub
  • Contenedor de Docker en Docker Hub

Además, se ha añadido una aplicación en Angular 5 que nos permitirá gestionar todas las operaciones existentes. Para poder acceder al código:
